# Gym Management System

A management system designed for gyms, focused on handling memberships, clients, and payments efficiently.

## Features
- Client registration and management
- Membership tracking and expiration control
- Payment management and reporting
- Role-based access (administrator / user)

## Tech Stack
- C# / .NET
- Entity Framework Core
- SQLite
- REST APIs (Minimal API)

## Project Structure
- **GimnasioApp/** → Core application logic
- **GimnasioApp.Desktop/** → Desktop interface and resources
- **TestDB/** → Database testing
- **ConsoleTest/** → Console-based testing
- **ManualTest/** → Manual validation scripts
